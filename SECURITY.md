# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in FediSpace, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email: security@echelon4.space

Or use [GitHub's private vulnerability reporting](https://github.com/sk7n4k3d/fedispace/security/advisories/new).

We will acknowledge your report within 48 hours and provide a timeline for a fix.

## Scope

- Authentication and token handling
- Data storage security
- Network communication security
- Third-party dependency vulnerabilities
